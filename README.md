# Agent-Immo-IA-MCP
Développement d’un serveur FastMCP exposant le modèle d’estimation DVF existant sous forme d’outils (estimation, biens similaires, diagnostic marché). Création d’un agent LangChain capable d’invoquer ces outils pour répondre en langage naturel à des questions immobilières. Containerisation avec Docker et déploiement sur K3s.
